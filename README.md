# Numpy--and--Pandas
# 📊 NumPy and Pandas Basics

## 📌 Project Overview
This project demonstrates the fundamental concepts of **NumPy** and **Pandas** using Python. It covers array creation, indexing, slicing, mathematical operations, and DataFrame manipulation for data analysis.

---

## 🎯 Objectives
- Learn NumPy array creation and operations.
- Perform array indexing and slicing.
- Convert Celsius temperatures to Fahrenheit.
- Create and manipulate Pandas Series.
- Create and analyze Pandas DataFrames.
- Perform filtering, grouping, and updating data.

---

## 🛠️ Technologies Used
- Python 3.x
- NumPy
- Pandas

## 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/yourusername/NumPy_Pandas_Basics.git
```

2. Move into the project folder.

```bash
cd NumPy_Pandas_Basics
```

3. Install the required libraries.

```bash
pip install numpy pandas
```

4. Run the Python program.

```bash
python numpy_pandas_basics.py
```
